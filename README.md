# valence

Valence is an open source Stellar Soroban project for building smart contract infrastructure, a backend service, and a frontend dashboard.

## Open source contribution

Valence is now configured for community contributions with:

- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `LICENSE`
- GitHub Actions CI for Rust tests, Soroban contract builds, and frontend builds

## Continuous integration

A GitHub Actions workflow runs on push and pull request events for `main`.
It validates the Rust workspace, builds the `contracts/valence-core` wasm contract, and builds the frontend.
